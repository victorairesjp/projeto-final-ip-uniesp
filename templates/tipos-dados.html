Tipos de Dados

Python possui diversos tipos de dados fundamentais. Os principais são:

  • int: números inteiros (idade = 20)
  • float: números decimais (altura = 1.75)
  • str: textos (nome = "Maria")
  • bool: valores lógicos (True ou False)

Exemplos de Uso

idade = 30           # int
altura = 1.82       # float
nome = "Carlos"     # str
aprovado = True     # bool

Verificando o tipo de uma variável

print(type(idade))    # 
print(type(altura))   # 
print(type(nome))     # 
print(type(aprovado)) #

Conversão de Tipos

Você pode converter entre tipos usando funções como int(), float(), str():

nota = "9.5"
nota_float = float(nota)  # converte string para float
print(nota_float)

Tipos Compostos

  • list: lista de valores (frutas = ["maçã", "banana"])
  • tuple: tupla, semelhante à lista, mas imutável (coordenadas = (10, 20))
  • dict: dicionário, pares chave-valor (aluno = {"nome": "Ana", "idade": 20})
  • set: conjunto de valores únicos (numeros = {1, 2, 3})

Exemplo de uso de tipos compostos

lista = [1, 2, 3]
tupla = (4, 5, 6)
dicionario = {"a": 1, "b": 2}
conjunto = {7, 8, 9}
