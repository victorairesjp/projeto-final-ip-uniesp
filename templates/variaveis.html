Variáveis

Variáveis são usadas para armazenar valores em Python. Elas são criadas no momento em que você atribui um valor a um nome. Não é necessário declarar o tipo da variável explicitamente, pois Python é uma linguagem de tipagem dinâmica.

Sintaxe Básica

nome = "João"
idade = 25
altura = 1.75

O nome da variável deve começar com uma letra ou sublinhado (_), não pode conter espaços ou caracteres especiais (exceto _), e diferencia maiúsculas de minúsculas (idade e Idade são diferentes).

Boas Práticas de Nomeação

  • Use nomes descritivos: nota_final ao invés de nf
  • Evite palavras reservadas do Python (ex: class, def)
  • Use snake_case para múltiplas palavras: media_turma

Exemplo Completo

# Atribuição de variáveis
nome = "Ana"
idade = 20
altura = 1.68

# Exibindo valores
print("Nome:", nome)
print("Idade:", idade)
print("Altura:", altura)

Reatribuição

Você pode mudar o valor de uma variável a qualquer momento:

idade = 21  # Agora idade vale 21

Identação

Python usa identação para definir blocos de código. Exemplo:

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Note que o bloco do if está identado com 4 espaços (ou um tab, mas o padrão é 4 espaços).
