Estruturas de Controle

Estruturas de controle permitem tomar decisões e repetir ações em Python. As principais são:

Condicionais (if, elif, else)

idade = 18
if idade >= 18:
    print("Maior de idade")
elif idade >= 12:
    print("Adolescente")
else:
    print("Criança")

Note a importância da identação: o bloco de código dentro do if, elif e else deve ser identado (4 espaços).

Laço for

for i in range(5):
    print(i)
# Saída: 0 1 2 3 4

O for é usado para percorrer sequências (listas, strings, etc).

Laço while

contador = 0
while contador < 3:
    print("Contador:", contador)
    contador += 1

Comandos de Controle

  • break: interrompe o laço
  • continue: pula para a próxima iteração
for numero in range(5):
    if numero == 3:
        break
    print(numero)
# Saída: 0 1 2

Exemplo Completo

notas = [7, 8, 6, 9]
soma = 0
for nota in notas:
    soma += nota
media = soma / len(notas)
if media >= 7:
    print("Aprovado")
else:
    print("Reprovado")
