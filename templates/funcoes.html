Funções

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Elas ajudam a organizar o código e evitar repetição.

Sintaxe Básica

def saudacao():
    print("Olá!")

saudacao()  # Chamada da função

Funções com Parâmetros

def soma(a, b):
    return a + b

resultado = soma(2, 3)
print(resultado)  # saída: 5

Parâmetros Opcionais e Valor Padrão

def apresentar(nome, saudacao="Olá"):
    print(f"{saudacao}, {nome}!")

apresentar("Ana")  # Olá, Ana!
apresentar("João", "Oi")  # Oi, João!

Retorno de Valores

def quadrado(x):
    return x * x

print(quadrado(4))  # saída: 16

Documentação de Funções

def dobro(n):
    """Retorna o dobro do número informado."""
    return n * 2

help(dobro)

Indentação

O bloco de código de uma função deve ser indentado (4 espaços por padrão):

def exemplo():
    print("Linha 1")
    print("Linha 2")

Funções Anônimas (lambda)

dobro = lambda x: x * 2
print(dobro(5))  # saída: 10
